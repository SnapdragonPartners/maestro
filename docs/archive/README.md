# Archived Documentation

Everything in this directory is **history with no authority for any question** (ADR [0017](../adr/0017-v2-documentation-authority-and-lifecycle.md), status `archive`). These are v1-era specs, plans, TODOs, and notes whose decisions were either implemented — making the code the authority — or abandoned. They are preserved verbatim, filenames intact, moved here by Phase 0 item 11 (see the [manifest](../v2/phase_0/manifest_doc-reset.md)); original paths are in git history, and no inbound links are maintained.

Do not cite these documents for v2 design or v1 runtime behavior. For v1 behavior, the authority order is code and tests, then the FSM docs, then the `deprecated`-status documents still at `docs/` root. For v2 design, use the Accepted ADRs (`docs/adr/` 0017+) and `docs/v2/`.
